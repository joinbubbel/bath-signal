## Getting Started

Add this to your `Cargo.toml` to integrate with backend.

```toml
bath-signal = "0.1"
```

Or, run with:

```shell
git clone https://github.com/joinbubbel/bath-signal
cd bath-signal
cargo r --example axum
```
